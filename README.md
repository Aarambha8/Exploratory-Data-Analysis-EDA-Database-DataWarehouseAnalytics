# Exploratory-Data-Analysis-EDA-Database-DataWarehouseAnalytics
Project Overview

This project focuses on Exploratory Data Analysis (EDA) performed on a Data Warehouse built using a Star Schema. The analysis uses curated Gold-layer fact and dimension tables to evaluate sales performance, customer behavior, and product contribution.

The goal of this EDA is to validate the data model, understand business patterns, and generate decision-ready insights before moving into dashboarding or advanced analytics.
